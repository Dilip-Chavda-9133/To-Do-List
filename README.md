Python Console-Based To-Do List Application

Developed a command-line To-Do List application using Python to manage daily tasks.

Implemented core functionalities including:
Add tasks
Update existing tasks
Delete tasks
View all tasks

Used lists (data structure) to store and manage task data dynamically.

Designed a menu-driven interface with continuous execution using a while loop.

Applied conditional statements (if-elif-else) for user choice handling.

Enabled user input handling and validation for smooth interaction.

Structured the program using functions for better organization and readability.
